<h1 align="center">Aurora Editor</h1>

<br />

Aurora Editor is an innovative Integrated Development Environment (IDE) meticulously crafted in Swift, designed to deliver exceptional native performance on macOS. Envision all the capabilities of Xcode, but enhanced with multi-language support, seamless Git integration, and a suite of your favorite built-in development tools. Aurora Editor is built by the community, for the community, aiming to be the go-to choice for developers seeking a robust and efficient coding environment.

## üöÄ Why Choose Aurora Editor?

In the world of code editors, many modern solutions are based on Electron, resulting in significant system resource consumption and less-than-optimal performance. Aurora Editor stands apart as a native macOS application, providing lightning-fast speeds, minimal RAM usage, and smooth, responsive animations. It offers a development experience akin to Xcode, with the added flexibility of supporting multiple programming languages. Whether you‚Äôre a seasoned developer or just starting, Aurora Editor ensures a seamless and efficient workflow, making coding a delight.
